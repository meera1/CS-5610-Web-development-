Click to slide down panel













Documentation 

      
        
  In this experiment I wanted to use jquery events like slide and toggle and animation

Code Sinppet

 
    <script>
        $(document).ready(function () {
            $("#flip").click(function () {
                $("#panel").slideToggle("slow");
            });
            $("button").click(function () {
                $("#div1").animate({
                    left: '250px',
                    opacity: '0.5',
                   
                });
            });
        });
    </script>
