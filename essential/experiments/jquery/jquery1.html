Experiment for learning jquery

Click the button below n see me disappear and then appear again

Documentation 

      
        
        In this experiment I wanted to use jquery. Basic selectors, for centain 
            elements as well as elements having certain class or id's

Syntax


        Basic syntax is: 
             $(selector).action()

        A $ sign to define/access jQuery
        A (selector) to "query (or find)" HTML elements
        A jQuery action() to be performed on the element(s)

Code Sinppet

 
    <script>
    $(document).ready(function () {
        $("button").click(function () {
            $("p").hide();
            $("#test").hide();
        });
    });
    </script>
