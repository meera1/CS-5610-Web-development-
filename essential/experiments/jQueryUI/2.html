    In this experiment, I have used JQueryUI. 
    What is JQuesryUI?
    jQuery UI is a curated set of user interface interactions, 
    effects, widgets, and themes built on top of the jQuery JavaScript Library.
    Whether you're building highly interactive web applications or you just
    need to add a date picker to a form control, jQuery UI is the perfect choice.
    
    I have used the 'draggable' and 'droppable' interactions availble. 
    I used bootstrap to make my page more spohisticated and a good feel, 
    Using these functionalities I have made a shopping cart for selecting items user wants to buy.
    In the java script, I also wrote another function for displaying an alert box on clicking the button.
