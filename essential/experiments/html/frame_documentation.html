In this experiment I wanted to make use of frames and develop a page where the user clicks on some links in one frame and the link opens up in another frame, all on the same page. All we have to do is use 'frame' and 'frameset' tags. The following is a snippet of my code: 

        <frameset border="0" frameborder="0" framespacing="0" rows="20%, *">
        <frame src="frames.html" />
        <frameset border="0" frameborder="0" framespacing="0" cols="25%,*">
        <frame src="frame_left.html" />
        <frame name="content" src="frame_body.html" />
        </frameset>
        </frameset>
