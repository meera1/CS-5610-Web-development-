EXPERIMENT 1



Drag image from one rectangle to another and back




Documentation

In this experiment I wanted to make elements draggable. I used images for this, setting its draggable attribute to true 

 "img draggable="true"

Then in the script tag, I defined function for the drag drop events

 
function allowDrop(ev) {
    ev.preventDefault();
}
function drag(ev) {
    ev.dataTransfer.setData("text", ev.target.id);
}
function drop(ev) {
    ev.preventDefault();
    var data = ev.dataTransfer.getData("text");
    ev.target.appendChild(document.getElementById(data));
}
