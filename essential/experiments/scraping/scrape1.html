Experiment 37: Web Scraping using AngularJS and NodeJS

Documentation 


    I will be working with web scraping in my project and so this expreiment is based on 
    using AngularJS and NodeJS to build a basic application that scrapes data from the given
    url. For scraping I will be using 'cheerio js' and 'request js' modules that I have used 
    in my previous NodeJS experiments
    
    What we will do in this experiment is that we will ask the user to enter the url and 
    using that url through the angular app, we will generate a http request to the server 
    where the above mentioned modules would scrape data.
