Code Snippet

        

    Below you can see the script that handles the onclick event on the paragraph tag in the body,
    Once it is cliked, you can see the json object as defined in the script
    -------------------------------------------------------------------------------------------------------------------
    
        The HTML code:
            
                
             <p class="p5" id="demo" onclick="displayJson();">Click here to see more!!!<p>


    -------------------------------------------------------------------------------------------------------------------
     
        The Script:

            <script>
            function displayJson() {
                var text = '{"name":"Meera Udani","street":"75 Saint Alphonsus St","phone":"1234567890"}'

                var obj = JSON.parse(text);

                document.getElementById("demo").innerHTML =
                obj.name + "<br>" +
                obj.street + "<br>" +
                obj.phone;
            }
                </script>
