Experiment 40: Web Scraping using AngularJS and NodeJS and Displaying in Tabular Form with Buttons for Selection and Removal

Documentation 


    
    What we will do in this experiment is that we will ask the user to enter the url and 
    using that url through the angular app, we will generate a http request to the server 
    where the above mentioned modules would scrape data. Along with this, which is 
    done in the previous exrepiment, we will be rendering the pure html that we 
    receive from the scraping module, and display in tabluar format in the front end.
    
    In this experiment, along with the above, we will add buttons to the tabular data
    such that the user can select the data he likes which will be displayed on the 
    same page by the angular application. 
    Apart from selecting the data a user likes, he will also be able to remove the selected 
    ones, which can be done by clicking the 'x' button besides his picks.
