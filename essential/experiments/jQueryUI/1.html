    Errors:
    Also I encountered an error "Uncaught TypeError: undefined is not a function"
    The resone for this error was that I was importing my jquery files in the wrong order
    The order should have been:
    jquery.css
    jquery.js
    jquery-ui.js

    Another reason I forund that this error occurs is that in the the script functions, 
    there is a ; missing or quotes, or parenthesis. Well in general syntax error.
    So the basic syntax should be something like:

    $(funtion (){
      // use $ code here ending with a ;
    })
