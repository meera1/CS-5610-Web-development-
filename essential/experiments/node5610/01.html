Experiment 26: Node Basics

Documentation 


    Kindly check 'serverApp.js' for this experiment which has a link at the bottom of this page    
    In this experiment, I started NodeJS, learnt the basics of nodejs.
    Wrote a server app which listens to port 3000 to a particular 
    url configured in the code and responds accordingly.
    So the step you need to follow are:
    - Open the browser
    - type in the url: http://localhost:3000/bye.html
    The result you get depends on the different strings you type in which can
    be found in 'serverApp.js'

Code Snippet


var express = require('express')   //imports
var app = express() //instance of expresss
app.get('/', function (req, res) {
    res.send('Hello World')
});
app.get('/bye', function (req, res) {
    res.send('Good Bye World')
});
app.listen(3000)
