    In this experiment, I have started a new topic called AngularJS

    What is AngularJS?
    AngularJS is a JavaScript framework.
    It can be added to an HTML page with a <script> tag.

    AngularJS works similar to the MVC frammework. It also has a 
    controller alson with directives, modules and expressions

    Directive: It is a marker on the HTML tag indicating it to reference some java script coode
    
    Modules: Every thing works like seperate modules or classes and thus can be resuable 

    Expressions: The AngularJS expressions are specifird in two curly braces, and thus 
                 on encountering them, it evaluates the content within them and displays the result 
                 eg. {{3+4}} output would be: 7


    So in this experiment, I just got my self aquianted with the new framework and libraries.
    We have to write two files, on is a javascript file having the AngularJS code and the other 
    a HTML file having the directives.
