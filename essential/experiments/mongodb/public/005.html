    Kindly check 'server.js' for this experiment which has a link at the bottom of this page. 
    I have worked on the previous experiment and built this one. 
    In this experiment, I used Angular application such that it acts as the middleware controller 
    that generates a http request to the server which is 'server.js' in this case and displayes 
    it, i.e. 'index 2.html' 
    In the previous experiment we ddeleted data from the Db. In this one we will be able to
    add new ones to the Db as well.
